## 폐지 종목 방어 정책

### 다층 방어 구조

| 층 | 방어 수단 | 담당 모듈 | 트리거 시점 |
|----|-----------|-----------|-------------|
| 1차 | F-Score 4점 필터 | `factors/quality.py` | 분기 스크리닝 |
| 2차 | 폐지 임박 일일 알림 | `monitor/risk_guard.py::check_delisting_imminent` | 매일 16:30 |
| 3차 | 분기 리밸런싱 자동 교체 | `strategy/rebalancer.py` | 분기말 리밸런싱 |
| 보조 | KRX 관리종목 감지 | `monitor/risk_guard.py::_check_delisting` | 장중 30분 간격 |
| 4차(opt-in) | 폐지 임박 자동 매도 | `monitor/risk_guard.py::execute_delisting_auto_sell` | 매일 09:30 |

### 수용된 한계

2017-2024 백테스트 기준:

| 지표 | 값 |
|------|-----|
| 폐지(failure) 노출 리밸런싱 | **1/32회 (3.1%)** |
| 단일 사례 포트폴리오 영향 | -5.00% (동일가중 20종목 × -100%) |
| 연간화 CAGR 영향 | **-0.64%p** |
| 사례 | 2017-Q2 005620 대성합동지주 |

이 수준의 노출은 **수용 가능** — Baseline 전략의 CAGR +18.76%에서 -0.64%p는 비용 대비 방어 효율이 맞는 범위.

### 검토했으나 미채택된 방어

2026-04-15 영향 평가 결과 전부 부적합 판정 (상세: `docs/reports/`):

| 방어 수단 | ΔCAGR | 005620 회피 | 판정 |

| `halt_history_filter_enabled` | -0.06%p | ❌ | ❌ 효과 없음 |
| **세 장치 통합** | -13.38%p | ✅ | ❌ 최악 |

### 의사결정 원칙

새로운 방어 장치 도입 시 **모든** 조건을 충족해야 함:

1. **CAGR 손실 ≤ -1%p** (8년 백테스트 기준)
2. **다음 중 하나 충족**:
   (a) 폐지 회피 효과 명확 (최소 005620 유형 배제 가능), OR
   (b) Alpha 동시 개선 (ΔCAGR > 0 AND ΔSharpe > 0) AND 종목 품질 sanity 통과
3. **Sharpe 하락 < 0.10**
4. **Baseline 대비 종목 겹침률 ≥ 90%** (유니버스 과소 차단)
   ※ 예외: 섹터 분산 등 종목 교체가 목적인 변경은 위양성 분석으로 대체 평가
   (위양성 평균 수익률이 미미하면 채택 가능)
5. **백테스트 8년(2017-2024) + 분기 구간별 안정성 확인**

위 조건을 하나라도 미충족하면 채택하지 않음.

## 전략 프리셋 확정 (v2.0)

### 핵심 추천: 프리셋 A (V70M30 + Vol70)

- `factor_weights: V=0.70, M=0.30, Q=0.00`
- `value_weights: PBR=0.50, PCR=0.30, DIV=0.20`
- `min_fscore=4` (필터로만 유지, 복합 스코어 미포함)
- `volatility.max_percentile=70` (상위 30% 고변동성 제외)
- `trailing_stop_pct=null, max_drawdown_pct=null, vol_target=null`

2017-2024 백테스트: CAGR +18.76%, MDD -30.92%, Sharpe 0.542

### 프리셋 변경 원칙

- 실전 운용 중 프리셋 변경은 **분기 리밸런싱 경계**에서만
- 프리셋 전환 시 반드시 Walk-Forward 재검증
- 전략 전용 키 (STRATEGY_ONLY_KEYS)는 금액 프리셋이 덮어쓸 수 없음

## 리밸런싱 정책

| 항목 | 값 |
|------|-----|
| 주기 | 분기 (3/6/9/12월 마지막 영업일) |
| 신호 계산 | 분기말 15:35 |
| 주문 체결 | 다음 영업일 시가 |
| 턴오버 상한 | 50% (프리셋 A 기준) |
| 재시도 | 최대 2회 (30분/60분 간격) |

시스템 리스크(코로나/전쟁/금융위기) 감지 시 프로그램 아닌 **운용자가 판단하여 수동 현금화**.
복귀는 시장 안정화 확인 후 다음 분기 리밸런싱에서 자동 재진입.

### S4: 금융주 제외 + 섹터 분산 (2026-05-13 채택)

금융주 56종목 자동 제외 (종목명 휴리스틱 기반) + 단일 섹터 최대 4종목 제약.

| 지표 | Δ (vs 금융주 제외 baseline) |
|------|---|
| CAGR | +0.44%p (6.36% → 6.80%) |
| Sharpe | +0.019 (0.240 → 0.259) |
| HHI (평균) | 2025 → 1191 (41% 감소) |
| 최대 섹터 비율 | 36.4% → 20.0% |
| 고유 섹터 수 (평균) | 9.7 → 11.5 |
| 종목 겹침률 | 73.4% |
| 위양성 평균 수익률 | +1.07% (n=101) — 미미 |

채택 근거: CAGR/Sharpe 동시 개선 + 분산 효과 강력 (HHI 41% 감소).
POLICY #4 (겹침률 ≥ 90%) 미달이나, 섹터 분산은 종목 교체가 목적이므로 예외 적용.

예외 사유: 겹침률 73.4%는 동일 종목 수(20개)를 유지하면서 섹터 간 교체한 결과.
위양성 평균 +1.07% (미미) + 중간값 -1.16% (정상 차단)으로 alpha 파괴 아님.
Step 3 본실험(겹침 70.7%, 위양성 +18.15%)과 달리 분산 목적의 건전한 교체.

상세: `docs/reports/sector_diversification_s4_analysis.md`
