# 생존자 편향 보정 분석 리포트

**기간**: 2017-01-01 ~ 2024-12-31  
**전략**: V70M30 + Vol70 (A 프리셋)  
**분석 기준일**: 2026-04-15  
**n_stocks**: 20  

## 요약

- 총 분석 리밸런싱 수: **32회**
- 폐지(failure) 종목 포함 리밸런싱: **1회**
- 누적 폐지 노출 건수: **1건**
- 누적 추정 손실 영향: **-5.00%** (동일가중, -100% 가정)
- 리밸런싱당 평균 영향: **-0.16%**

## 방법론

1. 분기 말 영업일마다 `MultiFactorScreener.screen()`으로 선정 종목 계산
2. `delisted_stock` 테이블에서 category='failure' 건수 추출
3. 선정 종목 중 다음 리밸런싱 전에 폐지된 종목 식별
4. 각 폐지 포지션을 동일가중(1/n_stocks) × -100% 손실로 추정
5. 실제 영향은 폐지 직전 거래정지·감자 등 추가 손실로 더 클 수 있음
