## 무너지는 지점 (Break-even)

- Sharpe < 0 지점: > 50bp (범위 내 미도달)
- CAGR < 0% 지점: > 50bp (범위 내 미도달)
- CAGR < KOSPI Buy-and-Hold(2.14%) 지점: > 50bp (범위 내 미도달)

→ **50bp까지 전략이 무너지지 않음.** 실거래 예상 비용(~20bp)의 2.5배에서도 KOSPI 초과수익 유지.
