## 4. 결론

### D 시나리오 (+30%→50% 익절)의 개선은 실질적인가?

시나리오 D가 CAGR +0.56%p, Sharpe +0.020으로 양쪽 모두 미미하게 개선되었으나,
**도입을 권고하지 않습니다.** 이유:

1. **Sharpe 차이 0.020은 통계적 노이즈 수준** — 8년간의 백테스트에서 이 정도 차이는
   데이터 기간을 1~2개월 이동하면 역전될 수 있는 수준
2. **MDD 개선 +3.14%p는 의미 있으나**, 여전히 -52.57%로 Baseline -55.70%과
   절대 수준이 유사 (둘 다 50% 이상 낙폭)
3. **익절 414회 발동** — 분기당 약 13회의 추가 매매가 필요하며,
   실전에서는 일별 모니터링 인프라와 장중 주문 체계가 추가로 필요
4. **놓친 수익 -3.30%** — 익절한 종목이 분기말까지 평균 3.3% 추가 하락했으나,
   31.2%의 경우에는 오히려 추가 상승을 놓침

### B, C, E 시나리오

- **B (+15%→50%)**: CAGR -0.77%p, Sharpe -0.029 → **명확히 악화**. 낮은 임계값은
  모멘텀 종목을 너무 빨리 매도하여 수익을 훼손
- **C (+20%→50%)**: CAGR -0.46%p, Sharpe -0.017 → **악화**
- **E (+20%→100% 전량)**: CAGR +0.19%p, Sharpe +0.007 → **노이즈 수준**.
  전량 익절 후 현금 유휴 기간의 기회비용이 큼

**현재의 분기 리밸런싱 전략(Baseline)을 유지합니다.**

부분 익절은 이론적으로 리스크 관리 수단이지만, 본 전략에서는:
- 분기 리밸런싱 자체가 자연스러운 이익 실현 역할을 수행

- 밸류 전략 특성상 저평가 종목의 장기 보유가 핵심 — 조기 익절은 이에 반함
