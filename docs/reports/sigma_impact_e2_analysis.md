| 지표 | A (σ=0.01 고정) | B (종목별 σ) | Δ |
|------|----------------|-------------|---|
| CAGR    | 6.28%   | 6.14%   | -0.14% |
| MDD     | -52.86% | -52.97% | -0.12% |

CAGR 차이 -0.14%p: 예측 범위(±0.1~0.2%p) 이내. 비용 모델 정확도 향상이 주목적.

## 종목별 일일 σ 분포 (기준: 20241230, N=2,446)

| 통계 | 값 |
|------|-----|
| 평균 (mean)    | 0.0525 (5.25%) |
| 중간값 (p50)   | 0.0297 (2.97%) |
| 하위 25% (p25) | 0.0209 (2.09%) |
| 상위 25% (p75) | 0.0420 (4.19%) |
| 최솟값 (min)   | 0.0013 (0.13%) |
| 최댓값 (max)   | 4.2837 (428%)  |

**핵심 관찰**: σ=1% 고정은 중간값(2.97%)의 절반도 안 됨 → 대부분 종목에서 시장충격을 과소 추정.
S5 선정 종목(KOSPI 대형·우량주)의 σ는 p25~p50 구간(2.1~3.0%)에 집중.

## 결론

- 종목별 σ 적용은 비용 모델 정확도 향상이 목적 — CAGR 변화는 -0.14%p (미미)
- KOSPI 선정 종목(저변동 우량주) 기준 σ_daily ≈ 2~3% >> 1% 고정
- 대형주: σ < 1% → (해당 없음, 실제 대형주도 1%↑)
- 소형주/고변동: σ > 1% → 더 정확한 고비용 추정
- S5(inverse-vol)와 시너지: 저변동성 종목에 높은 비중 + 낮은 시장충격 비용
- CAGR 소폭 감소(-0.14%p)는 비용 모델 보수화에 따른 자연스러운 결과
